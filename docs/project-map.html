The drill economy — grinds scenarios, sharpens the model. Every drill provenance-traced to a real game. M3 flex slot: seed drills from the 34 never-cast blind-spot cards.

Ante certified

The luck ledger — accounts what chance took and gave. AIVAT ledger certified (ADR-0014, re-certified at the omniscient tier); corrected reads standard in the arms eval (~5% variance cut).

Initial scope: 1v1 Commander (40-life ruleset), curated pool cf2ca6ba — 1,701 cards from 113 competitive Duel Commander decklists (ADR-0018: DC supplies the pool, not the rules; breadth scales set-by-set after core features).

How the pieces connect

Two invariant boundaries: the engine adjudicates every claim a learned component makes (the realizer enforces legality, never heuristic judgment), and the information set is enforced in exactly one leak-tested Python transform — the same featurization code path serves training and play (tests/test_serve_parity.py).

Night session: rollout contract CERTIFIED under the BC policy — Obs per-game wire sessions let forks feed the model server with the store off; forkcheck -grpc runs the full gate: smoke 10/10 clean, twin 40/40 = 100% determinism under the model, divergence 2.5% (the residual copy-gap class only). D2 — SA-level action schema + BC retrain done 07-11 — masked labels 0.004%, not the ~31% priced · agreement parity while naming the exact SA

The D8 order rung guessed 31–36% of casts (2.5× the corpus estimate): the largest known winrate leak, and RL needs SA-level action identity anyway. The label story collapsed in our favor (full-corpus sweep, 6.11M casts): 99.92% resolve exactly, masked = 0.004% — not the ~31% priced. The "ambiguity" was identical-string duplicates; collapsing identical (host, SA) candidate pairs makes the label exact (the executor keeps first-fit for the engine-side tie). Pointer key += learned SA descriptor over a pinned 33,691-string vocab (39.5M params); nested-window history fix rode along at measured-zero distribution shift; serve path needed zero Java changes (exact option index + host_level=false → the ladder degenerates to shape→pay). Retrain finals (774K steps, 15.1h): honest 0.9748 SA-basis / 0.9751 host-basis vs the 0.9758 host-level record — parity within run noise, with the model now naming the exact SA (nonpass 0.9247 on the harder basis; valpair ≥ val throughout; fresh δ=+2.180 → honest 0.9837). d2-sa/last.pt = the D3 candidate.

D3 — Winrate arms reprice done 07-11 — statistical tie: the order-rung mass was action identity, not winrate · order rung retired

Same 200 valpair pairs + seeds as D8 (paired): δ=0 0.4625 ± 0.0249 vs D8's 0.4675 — statistical tie; the order-rung mass did not cash out as winrate (kind-prior first-fit was a good proxy, and the exact-SA answer forfeits the ladder rescue: veto 7.8%→10.0%). Order rung retired in play (rungs = single+modal only); 800/800 decisive, 0 crashes, 0 fallbacks. What D2/D3 bought is the RL prerequisite — model-owned action identity at zero agreement cost. Closed with D2 by ADR-0012; new leak candidate = the veto class (~10% of cast attempts).

D4 — Ante certification + critic closed 07-13 — ledger certified zero-mean · winner-label poisoning found+fixed (ADR-0013) · critic +12pp mid-game AUC

Ante v1.1 ledger (AIVAT via observation surgery, zero Java changes) certified on both distributions (ADR-0014): 12,800-game mirror raw 0.4953 ± 0.0044, all classes zero-mean after the ledger's own test caught the re-deal choice-contamination; fitted-β convergence strictly faster than raw. The shakedown found corpus-wide label poisoning (ADR-0013): end.winner wrong ~50% of games — every value head to date learned seat noise. Fixed fork-side + reader-side (no regeneration); corrected labels lifted the frozen trunk to final-turn AUC 0.989 (d4-valuefix). Closed by ADR-0015: d4-critic-fullvis = critic of record (§4 asymmetric input): BCE 0.4541 / AUC 0.8658, +12pp over masked in the 9–16 band — clears the re-based gate by 12 points. Rollout-label machinery certified + priced (pilot: labels out-rank the critic in every band, +5.5pp at 5–8; ~17 pos/h/worker at K=8) — scale-up deferred behind server micro-batching. Ante re-certified at the omniscient tier: var cut 0.6%→4.8%, β̂→0.70, β=1 AIVAT now variance-reducing.

D5 — Combat constructs closed 07-14 — combat labels driven to 0 violations corpus-wide · arms tie as predicted · D6 unblocked

Factorized per-creature heads (attack logit + count + target pointer; block pointer over attackers∪none), labels via the combat-bounded obs join (the 145 corpus block violations = one overshoot class, driven to 0/113,591 games; a violation-free variant caught by loader-vs-measure exact reconciliation). d5-combat/last.pt = checkpoint of record: attack row 0.954 / target 0.993, block row 0.86; honest −0.77pp vs d2-sa (multi-task cost, priced at zero by the arms); value BCE 0.4531 on corrected labels. Executor: entity-ref AttackMap/BlockMap, engine-legality realizers with engine-authoritative repair (attacks self-validate in tiers; blocks forced-add — the engine never validates AI blocks). Arms tie as predicted: 0.4550 ± 0.0249 vs D3's 0.4625; drop rates 0.18%/3.7%, 0 forced/fallback. D6 data loop verified on model games. Closed by ADR-0016.

D6 — RL fine-tune done 07-17 — loop built · run-1 +3.0–3.8pp (≤1.7σ) · run-2 entropy collapse → guards (ADR-0017) · run-3 guarded chain surpasses the teacher (ADR-0019)

V-trace self-play machinery end-to-end per d6-vtrace-loop.md: fleet measured 1,146 g/h w=8 both-seats under micro-batching (target 1,000); seeded Gumbel-max sampling (batch-composition-independent by scatter-padded per-decision noise) + behavior-policy mu.jsonl keyed (g, s), zero Java changes; composite_logp = one body for the μ recompute tripwire / π ratios / PG term; V-trace targets unit-tested (caught a terminal double-count pre-training); synchronous loop driver (generate → ingest → train → swap) with the §6 anomaly monitor + arms cadence. 2-iteration smoke closed the loop: iteration 1 served the RL-trained ckpt, 0 tripwire violations, critic tracks realized reward.

d6-run1 (07-15 overnight): 12 iterations × 480 games ran to completion, zero unhandled failures — entropy/KL/critic monitors quiet throughout; first contact fixed 3 real bugs (mu-join chimera on diverged re-issues, segment-graph OOM, torn log lines; iterations now phase-idempotent). Arms: final ckpt 0.478 ± 0.011 on 2,000 games, paired +3.76pp ± 2.22 vs the D5 arm (t=1.69) — improves, not yet outside noise; done-when #4 open. Run-2 levers: the veto drift (14→19% under RL pressure — serve-time re-ask promotes to the front) and lr/iteration scaling (cumulative KL from init only ~0.05).

Run-2 prep (07-15 session 2): re-ask-on-veto BUILT and certified (design §6b; fork 64c6322f35: vetoed priority decisions re-issue with the vetoed candidate removed, fresh seq + μ record per attempt, retry_of wire mark; smoke 644 re-asks/40 games, tripwire 0) + Ante-ledger corrected reads folded into the standing arms eval (arms_report --ante, seat-flipped fitted-β merge). Arms re-baselined under the new environment: d5 0.4789 ± 0.0244 / rl1 0.5083 ± 0.0246 corrected — the RL ckpt's first read at parity with its teacher; paired rl1−d5 +3.00pp ± 1.93 (t=1.55), consistent with run-1. Re-ask alone worth ~+2pp to the BC arm; ~10% of vetoes rescue into casts, the rest become model-chosen passes the reward can see.

d6-run2 (lr 3e-5 × 16) collapsed overnight — caught at iter 6, diagnosed to the ground (ADR-0017): entropy-bonus runaway under near-zero mirror-self-play advantages (pg ±0.002 all run; the bonus was the only persistent gradient, lr its throttle — run-1's drift was the same force at ⅓ speed). Bonus retired → hinge floor; driver halt-guards (KL/entropy/veto triplines) promote to v0; anomaly rule two-sided. Run-3 = first guarded run, init from the run-1 ckpt. Run-3 (07-16→17, guarded: lr 2e-5, hinge floor, -reask): 13 healthy iterations, then the first live guard halt — veto tripline at 41.3%, ckpt rejected, ~10 min lost vs run-2's ~5h; entropy pinned all run (the hinge held). Strength grew throughout: iter-012 reads 52.2% ± 1.1 corrected at 2,000 games, +6.75pp paired over the D5 BC arm (t=2.92) — done-when #4 resolved (ADR-0019); the agent surpasses its teacher. Veto drift = the named open front, carried to M3 (commander-tax observation gap = the forensics-supported cause). Carried context: yield macros (§3b), ID dropout = the pre-Tutor lever, mode/allocation heads (1.2% / 0.03% mass).

Design invariants (standing law)

  • The engine adjudicates every claim any learned component makes.
  • Every LLM judgment is downstream-verified; LLMs never generate training truth.
  • Every drill is provenance-traced to a real game.
  • The value function is continuously audited against rollouts.
  • Detection is the engine's job; response is the model's.
  • The error-accounting queue is the spine everything reports to.
  • The model never sees the engine version — formats are rules to play to; versions are bugs not to learn.
  • Seed everything; deterministic replay. Provenance on all trajectories. Pinned fork per run.
