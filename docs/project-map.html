How the pieces connect

Two invariant boundaries: the engine adjudicates every claim a learned component makes (the realizer enforces legality, never heuristic judgment), and the information set is enforced in exactly one leak-tested Python transform — the same featurization code path serves training and play (tests/test_serve_parity.py).

M2 — Reinforcement learning open (plan: m2-rl-plan.md, seeded from ADR-0010)
  • 1. Fork-API hardening closed 2026-07-10 (ADR-0011) — stable single-step fork API (rollout prerequisite; the flagship upstream contribution, per ADR-0002). The forkcheck harness is the standing regression gate. Day one (07-10): static corruption 12%→0 (face-down exile leak + effect-card field wiring); trajectory divergence 50%→13.4% — root cause was GameCopier renumbering card ids (~10-line fix; residual = this-turn copy gaps, 67 repro seeds); twin determinism 99% (new -twin gate: two copies + same RNG snapshot must replay identically; the 1% tail = hash-ordered iteration in decision paths, isolated with the -bridge stress amplifier — bounds bit-exact replay only, doesn't bias rollout labels). Night session: rollout contract CERTIFIED under the BC policy — Obs per-game wire sessions let forks feed the model server with the store off; forkcheck -grpc runs the full gate: smoke 10/10 clean, twin 40/40 = 100% determinism under the model, divergence 2.5% (the residual copy-gap class only). Rollout pricing: copy 7 ms, fork→completion median 4.4 s (~70 positions/h/worker at K=8; server micro-batching = first D4 lever). Records: data/forkcheck/run-20260710{,-twin,-grpc}/. Late session: upstream PR #1 SUBMITTED — Card-Forge/forge#11203 (fixes re-authored on clean upstream, 3 differential regression tests each validated failing pre-fix, full suite 285 green; monitor for maintainer feedback each session). Anvil repo made public the same evening (LICENSE + README refresh). Closed by ADR-0011.

SA-level action schema + BC retrain retrain done 07-11 — the D8 order rung guessed 31–36% of casts (2.5× the corpus estimate): the largest known winrate leak, and RL needs SA-level action identity anyway. The label story collapsed in our favor (full-corpus sweep, 6.11M casts): 99.92% resolve exactly, masked = 0.004% — not the ~31% priced. The "ambiguity" was identical-string duplicates; collapsing identical (host, SA) candidate pairs makes the label exact (the executor keeps first-fit for the engine-side tie). Retrain finals (774K steps, 15.1h): honest 0.9748 SA-basis / 0.9751 host-basis vs the 0.9758 host-level record — parity within run noise, with the model now naming the exact SA (nonpass 0.9247 on the harder basis; valpair ≥ val throughout; fresh δ=+2.180 → honest 0.9837). d2-sa/last.pt = the D3 candidate.

Design invariants (standing law)

  • The engine adjudicates every claim any learned component makes.
  • Every LLM judgment is downstream-verified; LLMs never generate training truth.
  • Every drill is provenance-traced to a real game.
  • The value function is continuously audited against rollouts.
  • Detection is the engine's job; response is the model's.
  • The error-accounting queue is the spine everything reports to.
  • The model never sees the engine version — formats are rules to play to; versions are bugs not to learn.
  • Seed everything; deterministic replay. Provenance on all trajectories. Pinned fork per run.
