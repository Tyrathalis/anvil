## Done

- **Shakedown caught the pass trap on run one**: step-150 eval was pure always-pass (honest 0.90, nonpass 0.000) — the exact inflation the plan's metric discipline anticipated; nonpass-alongside-honest is not optional reporting.
- **pilot-run1: 20K steps / 5.12M windows / 21 minutes** (b=256 bf16, ~4,000 win/s — a full 29M-window epoch is ~2 h). Trajectory: nonpass 0.25 → dip 0.15 (pass-ward drift) → recovery → monotone climb through the decay phase. **Final val: honest agreement 0.944 / raw 0.945 / nonpass 0.708 (still rising at cutoff) / target-slot 0.951. valpair honest 0.944 — zero held-out-matchup gap.**
